## Background

The remaining v0.1.0 blockers are sustained external C fuzz evidence, realistic sanitized pcap corpus evidence, and tag-driven publication verification. The latest full sudo Docker RC validation remains recorded from 2026-07-08. On 2026-07-10, the local environment has Docker installed, but `sudo -n docker` requires an interactive password, so a non-interactive full sudo Docker RC refresh cannot be completed safely in this session.

## Goal
