## Background

`scripts/package_release.sh` generates `RELEASE_NOTES.md` inside release archives. The current file lists a few package contents and a minimal engine command, while `rc-check` only verifies that the file exists in the archive. The release artifact should carry a concise statement of package contents, quick verification, current v0.1.0 boundaries, and the release-candidate evidence expected for the archive.
