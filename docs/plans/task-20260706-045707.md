## Background

`scripts/rc_check.sh` already performs shell syntax checks with inline `bash -n` commands. Exposing the same check as a native Makefile target makes it easier to run directly, keeps release-candidate validation readable, and aligns with the current quality-gate pattern for docs, dependencies, coverage, and lint.

## Goal

Add `make shell-check` and make `rc-check` reuse it without changing runtime behavior.

## Scope

- Add `shell-check` to the root Makefile.
- Replace inline shell syntax commands in `scripts/rc_check.sh` with `make shell-check`.
- Update README, development guide, and changelog.
- Record validation results in this plan.

## Non-Goals

- No runtime code changes.
- No Docker behavior changes.
- No external shellcheck dependency.
- No broad shell style linting beyond syntax checks.
