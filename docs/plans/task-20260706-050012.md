## Background

NetSentry uses Python scripts for quickstart pcap generation and pcap sanitization. Release-candidate checks now include native shell syntax, documentation, dependency, coverage, fuzz, e2e, archive, and Docker gates, but Python script syntax is only exercised indirectly. A small native `python-check` target makes this validation explicit and reusable.

## Goal

Add `make python-check` using Python's standard `ast.parse` and include it in `make rc-check`.
